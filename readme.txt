readme.txt
スマブラSPの大会で使用するサムネイルを制作するジェネレーターです。

基本情報
インターネットに接続されている環境で使用してください。
1P、2P、大会情報に任意の文字列を入力し、右側にある緑色のプレビュー更新ボタンを押すと反映されます。
スライダーでファイター毎の倍率やX、Y位置を設定してください。文字情報と違い、リアルタイムで更新されます。
 (正常に動作していればスライダーの情報はファイター毎に保存されます。二回目以降そのファイターを指定する場合は前回の倍率・位置を参照します。）
 スライダーの情報は枠の中心を基準にしています。同ファイターの場合1P、2Pで設定しなおす必要はありません。
各文字情報はブラウザを更新する度デフォルトの状態に戻ります。都度入力して下さい。

各ボタンの説明
プレビュー更新
前述の通り、入力した文字列から画面を更新します。ファイターを変更した場合、スライダーの情報が先行して更新されます。ファイターが更新されるのを待ってください。

PNG書き出し
現在ジェネレーターのキャンバス上に表示されている画面を1920*1080で書き出します。ダウンロードフォルダに自動的に保存されます。

本戦用・予選用
クリックすることで別背景の画像に差し替えます。予選総当たり制を採用する場合や、BO5ラインを視覚的に区別したい場合に使用してください。



ファイターの名前は規則があります。間違えて入力すると参照できるデータが見つからないため反応しません。
以下命名規則
・日本語非対応。ファイターネームは英語入力
・大文字小文字は区別しない
・名前に『＆』や『and』を利用する、または2つ以上の英単語を使用するファイターは『_』を使用すること(例:young_link、banjo_and_kazooie)。
・名前に『.』を含む場合、省略する(例:mr_game_and_watch、rob)。
